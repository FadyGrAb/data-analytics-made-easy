The series structure

Mainly, I've split what I want to share into 6 categories that spans over the Data Engineering landscape:

  • Basic
  • Collection
  • Storage
  • Processing
  • Analysis
  • Visualization
In each category, I'll talk about 4 topics:
  • Theory
  • Open Source Software and tools
  • AWS
  • Mini-project(s)
I'll produce weekly content in the form of a video in Arabic on my Yet Another Data Guy YouTube Channel and a companion article in English at dev.to in a series with the name "data analytics made easy". So, make sure that you follow me there as well. Each category may have one or more mini-projects. The source code of those mini-projects will be in this Github repo.
